About This Application

Task Manager demonstrates relational database design using Peewee ORM.

Database Schema

Tables

  • user - User accounts
  • task - Todo items
  • tag - Task categories
  • task_tag - Association table (many-to-many)

Relationships

  • User.tasks - List of tasks assigned to user
  • Task.assignee - User who owns the task
  • Task.get_tags() - Tags associated with task
  • Tag.get_tasks() - Tasks with this tag
